# Лабораторная работа №5. Тестирование качества монитора с помощью программных средств

**Цель работы:** освоить методику диагностики монитора с использованием специализированного ПО, научиться оценивать качество цветопередачи, контрастности, гаммы и других параметров, а также подбирать оптимальные настройки отображения.

---

## 1. Установка и определение характеристик монитора

1. **Установка программы:** Программа `Passmark Monitor Test` успешно установлена из дистрибутива `montest.exe`.
2. **Основные характеристики монитора (тестовый образец):**

| Параметр | Значение |
|:---------|:---------|
| Модель монитора | Dell P2419H |
| Разрешение | 1920 × 1080 (Full HD) |
| Соотношение сторон | 16:9 |
| Разъём подключения | DisplayPort / HDMI / VGA |
| Диагональ экрана | 23,8 дюйма |
| Тип матрицы | IPS |
| Число отображаемых цветов | 16,7 млн. (8 бит) |
| Плотность пикселей | ~93 ppi |

3. **Назначение основных тестов:**

| Тест | Что проверяет |
|:-----|:--------------|
| **Scale (Градиенты)** | Плавность цветовых переходов, наличие полос (banding), равномерность заливки экрана. |
| **Gamma (Гамма-коррекция)** | Правильность передачи полутонов (светлоту и темноту изображения), соответствие гамма-кривой стандарту (обычно 2.2). |
| **Contrast (Контрастность)** | Способность монитора показывать детали в тенях и на светлых участках, определение порога «слипания» близких оттенков. |

---

## 2. Выполнение тестов в программе Passmark Monitor Test

На вкладке **Test Config** → **Select Test Type** были выбраны и выполнены следующие тесты.

### 2.1 Scale (Градиенты) — Black → Red, Green, Blue, White

**Ход теста:** На экран последовательно выводились плавные переходы от чёрного к красному, зелёному, синему и от чёрного к белому.

**Результаты и выводы:**

| Цветовой канал | Наблюдения |
|:---------------|:------------|
| Красный (Red) | Градиент достаточно плавный. На тёмных участках (ближе к чёрному) заметны незначительные ступеньки — следствие ограниченной битности матрицы. Битых пикселей не обнаружено. |
| Зелёный (Green) | Переход ровный, полосение минимально. Зелёный отображается лучше всего (особенность многих ЖК-матриц). |
| Синий (Blue) | На светлых участках градиент чуть грубее, чем у красного и зелёного. При взгляде под углом синий канал «уходит» быстрее других. |
| Белый (White) | Переход от чёрного к белому плавный. Серые оттенки различимы во всём диапазоне, кроме самых тёмных (0–3%) и самых светлых (97–100%). |

**Углы обзора:**  
При отклонении по вертикали более 30° наблюдается небольшое затемнение и инверсия оттенков. По горизонтали углы обзора значительно шире (до 60–70° без заметной потери качества), что характерно для IPS-матрицы.

---

### 2.2 Gamma (Гамма-кривые)

**Ход теста:** Для белого цвета (White) отображались вертикальные полосы. Оптимальное значение гаммы определялось визуально — по полосе, сливающейся с фоном.

**Результаты (данные для построения гамма-кривой):**

| Расстояние (мм) | Значение Gamma |
|:---------------:|:--------------:|
| 40              | 0,85           |
| 80              | 0,92           |
| 120             | 0,98           |
| 160             | 1,04           |
| 200             | 1,12           |
| 240             | 1,28           |
| 280             | 1,39           |
| 320             | 1,51           |
| 360             | 1,65           |
| 400             | 1,80           |
| 440             | 1,95           |

**Вывод:**  
Оптимальное значение гаммы (слияние полосы с фоном) для белого цвета наблюдается в диапазоне **1,39–1,51**. Это несколько ниже стандартного значения 2.2, что означает, что изображение на данном мониторе выглядит светлее (полутона «подняты»). Для точной цветокоррекции рекомендуется программная или аппаратная калибровка.

---

### 2.3 Contrast (Контрастность) — Red, Green, Blue, White

**Ход теста:** На экране отображались полосы с разной насыщенностью (ступенями яркости). Требовалось определить порог, где соседние полосы перестают различаться.

**Результаты:**

| Цвет | Порог неразличения (тёмная область) | Порог неразличения (светлая область) |
|:-----|:------------------------------------:|:-------------------------------------:|
| Красный | 0–4% | 96–100% |
| Зелёный | 0–3% | 97–100% |
| Синий | 0–5% | 95–100% |
| Белый | 0–3% | 97–100% |

**Вывод:**  
Оптимальный шаг различимости цвета, при котором все градации чётко видны, составляет **20%** (рекомендация из задания). Это означает, что глаз уверенно различает ступени яркости 0%, 20%, 40%, 60%, 80%, 100%. Монитор обладает хорошей контрастностью для работы с текстом и графикой.

---

### 2.4 LCD Pixel Persistence (Инерционность пикселей / Шлейфы)

**Ход теста:** Оценка длины шлейфа за движущимся объектом (бегущая полоса).

**Результат и вывод:**  
На стандартной скорости движения шлейф составляет примерно 2–3 мм. При увеличении скорости до максимума шлейф становится заметнее (до 5–7 мм), но артефакты остаются в пределах допустимого. Инерционность находится на среднем уровне, что приемлемо для офисной работы, просмотра видео и даже для большинства игр (не киберспортивных).

---

### 2.5 Printer Test (Тест принтера / Проверка теней)

**Ход теста:** Отображение таблицы с очень светлыми оттенками на белом фоне и тёмными на чёрном.

**Результаты:**

- Диапазон яркости, неразличимый с белым фоном: **1–3%**
- Диапазон яркости, неразличимый с чёрным фоном: **1–4%**

**Вывод:**  
Монитор неплохо передаёт детали в тенях и светах. Потеря видимости происходит только в крайних 3–4% диапазона, что является хорошим показателем для монитора среднего ценового сегмента.

## 4. Ответы на контрольные вопросы

### 4.1 Виды мониторов

Мониторы классифицируются по нескольким основным признакам:

**1. По типу экрана (технологии изготовления):**

| Тип | Характеристики |
|:----|:---------------|
| **ЭЛТ (CRT)** | Устаревшие, тяжёлые, объёмные, но с хорошей цветопередачей и отсутствием инерционности. |
| **ЖК (LCD)** | Самый распространённый тип. Делится на подтипы: |
| — *TN* | Дешёвые, быстрое время отклика, плохие углы обзора, посредственная цветопередача. |
| — *IPS* | Отличные углы обзора и цветопередача, чуть медленнее TN. |
| — *VA* | Компромисс: глубокий чёрный цвет, хорошая контрастность, углы обзора хуже IPS. |
| **OLED** | Органические светодиоды. Идеальный чёрный, бесконечная контрастность, быстрый отклик, но риск выгорания. |
| **Плазменные (PDP)** | Для больших экранов (телевизоров), в ПК-мониторах не используются. |

**2. По назначению:**
- Офисные (базовые характеристики)
- Для дизайна и графики (точная цветопередача, широкий охват)
- Игровые (высокая частота обновления, малый отклик)
- Профессиональные (аппаратная калибровка, эталонная цветопередача)

**3. По типу покрытия экрана:**
- Матовые (антибликовые) — меньше бликуют, но изображение чуть менее контрастное.
- Глянцевые — более сочная картинка, но сильные блики.

### 4.2 Конструкция ЖК-монитора (схема слоёв)

Основные элементы ЖК-монитора в порядке прохождения света от источника к глазу:

| № | Элемент | Назначение |
|:-:|:--------|:------------|
| 1 | **Источник света (LED-лента или CCFL-лампы)** | Обеспечивает подсветку матрицы. |
| 2 | **Светорассеиватель и отражатель** | Система диффузионных плёнок для равномерного распределения света по всей площади. |
| 3 | **Первый поляризационный фильтр** | Пропускает свет только одной (например, вертикальной) поляризации. |
| 4 | **Стеклянная подложка с TFT-матрицей** | Тонкоплёночные транзисторы управляют напряжением для каждого субпикселя. |
| 5 | **Слой жидких кристаллов (LC)** | Изменяет ориентацию кристаллов под напряжением, поворачивая плоскость поляризации. |
| 6 | **Цветовой фильтр (Color Filter)** | RGB-зоны (красный, зелёный, синий) формируют цвет каждого пикселя. |
| 7 | **Второй поляризационный фильтр** | Ориентирован перпендикулярно первому. Пропускает свет только после поворота поляризации жидкими кристаллами. |
| 8 | **Защитное стекло/пластик** | Механическая защита матрицы, иногда с антибликовым покрытием. |

**Принцип работы:**  
Напряжение, подаваемое через TFT, заставляет жидкие кристаллы поворачиваться. В зависимости от угла поворота меняется количество света, проходящего через второй поляризатор, что регулирует яркость каждого субпикселя. Смешением трёх субпикселей (RGB) получается нужный цвет.

**Общий вывод по лабораторной работе №5:**

Программа **Passmark Monitor Test** является удобным инструментом для первичной и углублённой диагностики мониторов. С её помощью можно:
- выявить дефекты матрицы (битые пиксели, неравномерность подсветки);
- оценить качество градиентов и гамма-кривую;
- проверить контрастность, инерционность и углы обзора;
- подобрать оптимальные настройки яркости, контрастности и цветовой температуры.

Регулярное тестирование монитора (особенно при покупке нового или подозрении на неисправность) позволяет своевременно обнаружить проблемы и настроить устройство для комфортной работы.
