# Практическая работа №7

## Тестирование компьютерной системы в программе AIDA64

**Цель работы:** ознакомиться с функционалом программы AIDA64 для комплексной диагностики и тестирования ПК, освоить методику мониторинга датчиков и проведения стресс-тестов.

**Оборудование:** персональный компьютер под управлением ОС Windows, программа AIDA64 Extreme Edition (установленная), лицензионный ключ.

1. Программа `AIDA64 Extreme Edition` успешно установлена на рабочий компьютер согласно инструкции.

## 2. Сравнительный анализ диагностического ПО

В ходе изучения интерфейса и возможностей AIDA64 был проведён сравнительный анализ с аналогичными программами для диагностики и тестирования.

| Программа | Назначение | Преимущества | Недостатки |
|:----------|:-----------|:-------------|:------------|
| **AIDA64 Extreme** | Диагностика, мониторинг, бенчмарки, стресс-тест | Самый широкий функционал: подробнейшая информация о железе и ПО, множество тестов, мониторинг датчиков в реальном времени | Платная (есть ограниченная пробная версия 30 дней) |
| **PC Wizard** | Диагностика и базовое тестирование | Бесплатная, хорошая информативность, есть базовые тесты производительности | Давно не обновляется (последняя версия 2015 г.), устаревший интерфейс |
| **HWiNFO64** | Диагностика и мониторинг | Бесплатная, очень детальные показания датчиков, поддержка новейшего железа | Нет встроенных бенчмарков, сложный интерфейс для новичка |
| **FurMark** | Стресс-тест видеокарты | Бесплатный, создаёт максимальную нагрузку на GPU, лучший для проверки стабильности видеокарты | Узкая специализация (только GPU) |
| **PassMark BurnInTest** | Комплексный стресс-тест | Профессиональный инструмент, тестирует все компоненты одновременно | Сложен для начинающих, платный, слабая диагностическая часть |

**Вывод по сравнению:**  
Программа **AIDA64 Extreme Edition** обладает наибольшим функционалом среди рассмотренных аналогов. В отличие от узкоспециализированных утилит (FurMark, BurnInTest), она сочетает в себе мощную диагностическую базу (детальные сведения о компонентах), подробный мониторинг (температуры, напряжения, токи, частоты) и обширный набор тестов для CPU, памяти, дисков и GPU. Это делает её оптимальным выбором для комплексной проверки ПК.

## 8. Ответы на контрольные вопросы

### Вопрос 1: Для чего необходимо выполнять стресс-тесты компьютерной системы?

Стресс-тесты нужны для проверки стабильности и надёжности работы компонентов ПК в условиях максимальной нагрузки. Основные задачи:

| Задача | Описание |
|:-------|:---------|
| **Выявление перегрева** | Проверка эффективности системы охлаждения (CPU, GPU, чипсет). Перегрев > 85–90°C указывает на проблемы. |
| **Проверка после разгона** | После оверклокинга стресс-тест показывает, стабильно ли работают компоненты на повышенных частотах. |
| **Диагностика скрытых дефектов** | Некоторые неисправности (ошибки памяти, дефекты чипов) проявляются только под нагрузкой. |
| **Проверка блока питания** | При максимальном энергопотреблении компонентов БП должен выдавать стабильное напряжение. |
| **Приёмочное тестирование** | После сборки или ремонта ПК стресс-тест подтверждает работоспособность системы. |

### Вопрос 2: Для чего используются программы диагностики компьютера?

Программы диагностики (например, AIDA64, HWiNFO64, PC Wizard) используются для:

| Назначение | Пример использования |
|:-----------|:---------------------|
| **Получение точной информации об оборудовании** | Определить модель материнской платы, версию BIOS, тип оперативной памяти, не вскрывая корпус. |
| **Мониторинг состояния в реальном времени** | Отслеживать температуры, напряжения, обороты вентиляторов, частоты процессора и видеокарты. |
| **Выявление «узких мест» производительности** | Определить, что является причиной тормозов: медленный диск, недостаток памяти или слабый процессор. |
| **Создание отчётов** | Составить детальный отчёт о конфигурации для апгрейда, ремонта или гарантийного обслуживания. |
| **Сравнение с эталонными результатами** | Сопоставить производительность своего ПК с аналогичными системами из базы программы. |

### Вопрос 3: Какая из известных Вам программ обладает наибольшим функционалом для тестирования ПК?

Наибольшим функционалом для тестирования ПК обладает программа **AIDA64 Extreme Edition**.

**Обоснование (подтверждено сравнительным анализом в п. 2):**

1. **Диагностика:** выдаёт самую подробную информацию обо всех компонентах (более 100 страниц данных).
2. **Мониторинг:** показывает температуры, напряжения, токи, мощности, обороты вентиляторов в реальном времени с графиками.
3. **Тесты производительности:** более 30 бенчмарков для CPU, памяти, дисков, GPU (Queen, PhotoWorxx, AES, Julia, Memory и др.).
4. **Стресс-тест:** мощный инструмент для проверки стабильности системы под нагрузкой (CPU, FPU, кэш, память, диски, GPU).
5. **GPGPU тестирование:** проверка вычислительной производительности видеокарты в неграфических задачах.

Таким образом, AIDA64 объединяет возможности узкоспециализированных тестов (как FurMark для GPU или MemTest для памяти) и мощную диагностическую базу, что делает её лучшим решением для комплексной проверки ПК.
